# Inspiration HTMLs

Public-safe HTML inspiration artifacts. Use these as visual/style references for prompts and generated assets. Do not treat them as site copy or source of factual content.

Subfolders:

- `beautiful-html-templates/` — generic public-safe template library.
- `brand-html-templates/` — rendered anonymized samples from local brand/style template sources; raw private names/templates are not published.
- `website-examples/` — anonymized website/landing examples with original names and private provenance removed.

Direct private/client/person-specific HTML is intentionally not published here.
