# Brand HTML Templates

Rendered, anonymized public-safe samples generated from local brand/style template sources.

These are **not** raw client/person/org templates. They exist because the original Pandoc-style templates render as blank/broken pages in a browser unless filled with sample content and CSS.

Use them only as secondary visual inspiration. Canonical `viggomeesters.com` style remains `STYLE.md` and `viggomeesters-htmls/`.
