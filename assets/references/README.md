# Reference Assets

Public-safe visual references for future ViggoMeesters hero/OG prompts.

Each reference should have:

```text
<slug>.png|jpg|webp
<slug>.prompt.md
<slug>.notes.md
```

The notes file should explain:

- source page or repo;
- creation route: where/how the image was made or received;
- whether the image is generated, supplied, or hand-authored;
- style traits worth preserving;
- public-safety review notes;
- whether it can be used as a direct style anchor.

Do not place private knowledge-base screenshots, client material, or internal-project images here.
